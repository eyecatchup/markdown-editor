Markdown Editor
---------------

Basic feature list:

 * Ctrl+S / Cmd+S to **save to file** 
 * Ctrl+Shift+S / Cmd+Shift+S to choose to save as **Markdown or HTML** file
 * **Drag and drop** a file into here to load it
 * Contents are saved in the URL so you can easily **share your notes**
